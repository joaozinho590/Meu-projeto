Projeto de Integração de Script Python com Sistema de Hardware

Introdução

Bem-vindo ao repositório do projeto de integração de um script Python com um sistema de hardware utilizando a biblioteca GPIO Zero! Este projeto foi desenvolvido com o objetivo de demonstrar como scripts Python podem ser utilizados para interagir diretamente com componentes de hardware, fornecendo uma interface simples e eficiente para controle e monitoramento de dispositivos físicos.

Descrição do Projeto
Neste repositório, você encontrará um script Python de teste, projetado para se comunicar com um sistema de hardware específico. A principal função deste script é ilustrar a forma como o Python pode ser utilizado para enviar comandos, receber dados e controlar o comportamento de dispositivos de hardware em tempo real.

Funcionalidades:
Interação com Hardware: Utilização da biblioteca GPIO Zero para controle de pinos GPIO em uma placa Raspberry Pi.
Controle de Dispositivos: O script controla um LED conectado ao pino GPIO 17 e monitora um botão conectado ao pino GPIO 18.
Detecção de Eventos: O script imprime mensagens no console sempre que o botão é pressionado ou liberado.
Ciclo de Ativação/Desativação: O LED alterna entre ligado e desligado a cada segundo em um loop contínuo.
